# Project Overview: Urban Heat Insights

## Executive Summary
Urban Heat Insights is an interactive geospatial analytics platform predicting Urban Heat Island (UHI) phenomena across Malaysia via Machine Learning and Mapbox mapping. It bridges the gap between raw Python satellite extraction and instantaneous visual React prediction outputs, aiming to guide city planners with statistical precision.

- **Tech Stack:** React (Frontend), FastAPI (Backend), Supabase (PostgreSQL), XGBoost/Random Forest (ML).
- **Structure:** Multi-part Architecture (Monorepo).
